# wcp-icon

Shows an icon from the css.gg icon set.

## Examples

```html
<wcp-icon name="smartphone"></wcp-icon>
```

### Set a custom size

```html
<wcp-icon name="laptop" size="24px"></wcp-icon>
```

## Properties

| Property | Attribute | Type                  |
|----------|-----------|-----------------------|
| `name`   | `name`    | `string`              |
| `size`   | `size`    | `string \| undefined` |
