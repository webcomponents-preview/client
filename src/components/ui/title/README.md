# wcp-title

Shows the application title and a logo.

## Example

```html
<wcp-title title="Web Components Preview">
  <img slot="logo" src="assets/icons/logo.svg" height="30px" />
</wcp-title>
```

## Properties

| Property | Attribute | Type     |
|----------|-----------|----------|
| `title`  | `title`   | `string` |

## Slots

| Name   | Description                         |
|--------|-------------------------------------|
| `logo` | Receives the logo image to be shown |
