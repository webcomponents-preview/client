# Code examples

It may have some `inline code` as well.

```js
const foo = 'bar';
```

```ts
const foo: string = 'bar';
```

```html
<body data-attribute="foo">
  <h1 class="title">Some headline</h1>
</body>
```

```css
body {
  background-color: #fff;
}
```

```scss
$color: #fff;

body {
  background-color: $color;
}
```

```bash
$ npm install
```

```json
{
  "name": "foo",
  "version": "1.0.0",
  "description": "Some description"
}
```

```yaml
name: foo
version: 1.0.0
description: Some description
```
